7. Design Guidelines

Rules for maintaining visual consistency with the Figma source of truth.

Do
  • Use exact Figma font sizes (13.83px, 39.927px, 55.59px) via design tokens
  • Use #030948 for cards on dark backgrounds
  • Use the 4px blue accent line as a section separator
  • Highlight key words in headings with #2469ff
  • Use 19px border-radius consistently on all cards
  • Use 3px solid #2469ff + blue glow for highlighted cards
  • Use Outfit for headings and body, Instrument Sans for labels/captions, Big Shoulders for numbers
  • Use the dark gradient linear-gradient(105.39deg, ...) for dark page backgrounds
  • Keep body text at Outfit Regular 13.83px, not Inter
Don't
  • Don't use Tailwind defaults like text-4xl or rounded-3xl instead of Figma tokens
  • Don't use bg-slate-950 or bg-slate-900 as dark backgrounds - use the gradient
  • Don't use backdrop-blur on metric cards (that's a different visual than Figma)
  • Don't use Inter for body text - Figma specifies Outfit
  • Don't scale up Big Shoulders numbers beyond 55.59px - that's the max in Figma
  • Don't mix border-2 and border-3 on highlighted cards - always 3px
  • Don't omit the 4px accent line from section headers
  • Don't use gradient text effects not present in Figma
